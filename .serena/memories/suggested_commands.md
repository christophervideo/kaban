# Suggested Commands

## Development

```bash
bun run dev          # Run the application
bun run build        # Build for production
```

## Package Management

```bash
bun add <package>    # Add dependency
bun install          # Install dependencies
```

## System (Darwin/macOS)

```bash
ls -la               # List files
git status           # Git status
git add . && git commit -m "msg"  # Commit changes
```
