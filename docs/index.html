Quick Start

Terminal 
# Install
git clone https://github.com/beshkenadze/kaban
cd kaban && bun install && bun run build

# Initialize a board in your project
kaban init --name "My Project"

# Add a task
kaban add "Implement feature X" --agent claude

# Launch the TUI
kaban tui

For AI Agents

Connect your AI coding assistant to Kaban via MCP (Model Context Protocol).

claude_desktop_config.json 
{
  "mcpServers": {
    "kaban": {
      "command": "kaban",
      "args": ["mcp"],
      "env": {
        "KABAN_PATH": "/path/to/your/project"
      }
    }
  }
}

Available MCP Tools

kaban_add_task Add new tasks
kaban_list_tasks Query tasks with filters
kaban_move_task Move between columns
kaban_complete_task Mark as done
kaban_update_task Edit task details
kaban_status Board overview

CLI Commands

kaban init Initialize a new board
kaban add <title> Add a task
kaban list List tasks with filters
kaban move <id> Move task to column
kaban done <id> Mark task complete
kaban tui Launch interactive UI
kaban status Show board summary
kaban mcp Start MCP server

Installation

Prerequisites

  • Bun runtime (v1.0+)
  • Task (optional, for easy install)

With Task (Recommended)

git clone https://github.com/beshkenadze/kaban
cd kaban
task install

Manual Installation

git clone https://github.com/beshkenadze/kaban
cd kaban
bun install
bun run build
# Add to PATH or create alias
alias kaban="bun run /path/to/kaban/packages/cli/src/index.ts"
